# 03 ClassLoading / Manifests / Index

本目录放的是 **“本章覆盖范围”与“落地证据清单”**，用于：

- 让新人知道本章要看哪些 runtime 文件（以及为什么）
- 支撑 Completion Review 做“覆盖率/断言证据链”验收

## 文件说明

- `files.yaml`
  - **用途**：本章“关键源文件清单”（人为维护）
  - **典型用法**：review/排障时，用它对照确认关键文件是否都被读过/写过 FileNotes
- `tree_inventory.txt`
  - **用途**：目录树快照（方便快速查找文件名）
